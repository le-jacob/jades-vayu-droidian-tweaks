# PRE-REQUISITIES
- Poco X3 Pro with Droidian (of course)
- Root access (need to install the service)

# HOW TO INSTALL
1. Open a terminal window (or ssh into the phone if enabled)
2. Download the service file
3. Move the service file into a place that you won't delete (i recommend: /root/.scripts/)
4. Run 'systemctl enable /root/.scripts/doubletapdisable.service' with root
5. Pray that it works
